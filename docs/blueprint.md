# **App Name**: AI Synapse

## Core Features:

- Reliable AI Source Aggregation: Automatically fetch and display the latest articles from official RSS feeds of leading AI companies (Anthropic, OpenAI, Google DeepMind, Meta AI).
- AI Discovery Platform Integration: Automatically fetch and display new AI-tagged content from Product Hunt, Hacker News, and GitHub Trending.
- AI-Powered Content Summarization: Use the Gemini API as a tool to generate concise summaries for each aggregated article or discovery item.
- Unified Content Display: Clearly display the article title, AI-generated summary, original source, and publication date/time for each item.
- Dynamic Categorization & Timeline: Organize content into 'Reliable Sources' and 'Discovery' categories, presenting all items in a chronological, new-to-old timeline.
- Custom RSS Feed Management: Allow users to add and manage additional RSS feed URLs to personalize their content dashboard, storing them in Firestore.

## Style Guidelines:

- Light color scheme to enhance readability for information consumption.
- Primary color: Deep indigo (#2929A3), reflecting intelligence and depth.
- Background color: Subtle off-white with an indigo tint (#EBEBF2), providing a clean and calm base.
- Accent color: Vibrant blue (#2692DE), for highlighting key information and interactive elements.
- Headlines font: 'Space Grotesk' (sans-serif), for a modern and tech-inspired look.
- Body text font: 'Inter' (sans-serif), chosen for its excellent readability and neutral, professional aesthetic in information-heavy interfaces.
- Use minimalist, crisp line-art icons that provide clear visual cues for sources, categories, and user actions, maintaining a modern and clean aesthetic.
- Employ a responsive dashboard layout with clear card-based presentation for each content item, optimizing for information density and scanability on various screen sizes.
- Incorporate subtle hover effects on clickable cards and smooth transitions for content loading and category switching to provide a fluid user experience without distraction.
